# Water Intake Tracker

A simple web-based water tracking application built with Python, Flask, HTML, and CSS. This app helps users calculate how much water they should drink throughout the day based on their personal hydration goal, schedule, and custom drinking intervals.

---

## Features

- Personalized water intake calculations
- Custom daily water goals in ounces
- Adjustable drinking intervals
- Progress/result tracker
- CSV logging system for saved progress
- Reset progress functionality
- Responsive web-based interface

---

## Technologies Used

- Python
- Flask
- HTML5
- CSS3
- CSV File Handling
